Slope measures how fast a function's output changes as its input changes. For a
straight line it is the constant ratio of vertical change to horizontal change, the
same number between any two points. It answers "per one unit of input, how much does
the output move, and in which direction".

Formally, the slope between points `(x₁, y₁)` and `(x₂, y₂)` is
`m = (y₂ − y₁)/(x₂ − x₁)` for `x₁ ≠ x₂`. A positive `m` rises, a negative `m`
falls, and `m = 0` is flat. For a line, `y = mx + c`.
