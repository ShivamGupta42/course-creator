A linear equation sets two expressions equal where the unknown appears only to the
first power. Solving means finding every value of the unknown that makes the
equality true, by applying the same operation to both sides until the unknown
stands alone. Each step must be reversible so no solution is gained or lost.

Formally, `ax + b = c` with `a ≠ 0` has the unique solution `x = (c − b)/a`. The
allowed moves are adding the same quantity to both sides and multiplying both sides
by the same nonzero quantity, which preserve the solution set.
