A set is a collection of distinct objects with no order and no repeats, and counting
asks how many objects a described set contains. Union, intersection, and complement
combine sets, and counting rules turn a description of choices into a number without
listing every case.

Formally, for finite sets `|A ∪ B| = |A| + |B| − |A ∩ B|` (inclusion-exclusion), and
a sequence of independent choices with `n₁, n₂, …` options has `n₁·n₂·…` outcomes by
the multiplication principle. Membership `x ∈ A` is the primitive relation.
