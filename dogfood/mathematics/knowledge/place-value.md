A written number is a sum of digits weighted by powers of the base. In base ten,
the digit in a column counts how many of that column's power of ten you have, so
`347 = 3·10² + 4·10¹ + 7·10⁰`. The number line places every real number at a
fixed position, with order and distance read off directly.

Formally, a base-`b` numeral `dₙ…d₁d₀` denotes `Σ dᵢ·bⁱ` with each digit
`0 ≤ dᵢ < b`. Distance between two points `a` and `b` on the line is `|a − b|`.
