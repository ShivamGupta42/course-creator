A variable is a placeholder that stands for any value from some set, and an
expression is a recipe built from variables, numbers, and operations. Replacing
the variable with a value and evaluating gives the expression's value at that
input. The same expression can be rewritten into equal forms using the field laws.

Formally, an expression denotes a function of its free variables; two expressions
are equal when they agree for every assignment, so `2(x + 3) = 2x + 6` holds for
all `x`. Substitution is the act of fixing a value and reducing to a number.
