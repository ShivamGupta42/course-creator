Worked Module 10. Started by re-testing the negative-coefficient linear equation
flagged last session: gave `−4x + 2 = 10`, and they got `x = −2` correctly this
time, talking through the sign aloud. Still slower than the positive case, so
linear-equations stays `shaky` for one more review, but the repair is holding.

Functions went in through the vending-machine framing: each button (input) drops
exactly one item (output), and a machine that sometimes gave two items for one
button would not be a function. That landed. The learner restated one-input-one-
output without prompting and gave their own example, a bus timetable mapping each
departure time to one bus.

Stumble: the vertical-line test. They could state it but applied it shakily on a
sideways parabola, first calling it a function, then catching the doubled output
when I asked them to trace `x = 4` upward. They need one more pass where they run
the test themselves on three graphs before it is automatic. Marked functions
`shaky`. Next session: drill the vertical-line test to solid, then start slope,
which the mission names as the first real goal. Did not start slope or limits.
